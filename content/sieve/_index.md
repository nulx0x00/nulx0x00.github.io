# NulX Sieve - PII Discovery Tool

High-performance Python tool for automated PII discovery across file systems, databases, and APIs.

## Features
- Multi-source scanning
- Pattern detection (regex + ML)
- Fast async I/O
- Minimal dependencies

## Quick Start
```bash
git clone https://github.com/pr0jzer0/nulx-sieve
cd nulx-sieve
pip install -r requirements.txt
python sieve.py scan --path /var/www
```
